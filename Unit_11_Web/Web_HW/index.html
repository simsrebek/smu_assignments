Summary: Latitude vs. X

The purpose of the project was to analyze the weather closer to the equator. Using Jupyter Notebook with multiple dependecies such as openweathermapy,citipy and the OpenWeatherMap API, I generated a cities list which contained a count of over 600 random cities /latitude and longitude. Data was formatted inorder to create the images for the analysis. Visualization of the of the analysis was best interpreted by scatterplots to show the relationship between latitude -temperature,humidity,cloudiness and wind.
